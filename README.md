# IPL-Match-Outcome-Predictor
Tools: Python  •  Pandas  •  NumPy  •  Scikit-learn  •  Matplotlib  •  Seaborn
▸	Conducted end-to-end Exploratory Data Analysis (EDA) on an IPL dataset to uncover patterns in team performance, venue impact, and batting/bowling trends using Matplotlib and Seaborn.
▸	Engineered relevant features and preprocessed raw data — converting NumPy arrays to structured Pandas DataFrames — to prepare a clean dataset for model training.
▸	Built and evaluated a binary classification model using Scikit-learn to predict whether a team batting first would win, achieving an accuracy of 57% and an ROC-AUC score of 0.506.
▸	Gained practical experience in the full ML pipeline: data cleaning, feature engineering, model selection, and performance evaluation on real-world sports data.
